# AI-Security-Research
Exploring the security of AI systems through adversarial machine learning, LLM penetration testing, prompt engineering attacks, model exploitation, and mitigation strategies.


---

AI is everywhere, and attackers know it. 


From poisoned training data to prompt injection, the threats targeting AI systems are real, evolving, and misunderstood. Whether you're defending an LLM-powered application or auditing an AI pipeline, 

THis Repo holds a collection of Research work aimed at building knowledge to  help one think like an AI security engineer.
